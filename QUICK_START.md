## â“ **Common Questions**

**Q: Do I need to create an account?**  
A: No! The app works with your device storage. No login needed.

**Q: What if internet goes down?**  
A: No problem! App works offline. Data saves locally and syncs when connection returns.

**Q: Can I see data on my phone and laptop?**  
A: Yes, if server is connected. Data syncs automatically. If not, each device keeps its own copy.

**Q: Is my data secure?**  
A: Data is stored on your device. If you export CSV, keep it safe like any sensitive file.

**Q: What if I lose my phone?**  
A: Data will be lost unless you backed it up. **Export CSV weekly to be safe!**

**Q: Can multiple people use the same device?**  
A: Yes, but they'll share the same data. Separate devices = separate data.

## ğŸ†˜ **Something Not Working?**

### **App won't load:**
1. Refresh browser (F5)
2. Clear browser cache (Ctrl+Shift+Delete)
3. Try a different browser
4. Check internet connection

### **Data disappeared:**
1. Check if you're on the same device
2. Try importing from exported CSV if you have one
3. Contact developer

### **Can't export CSV:**
1. Try "Copy CSV" first
2. If that works, use keyboard Ctrl+C to copy
3. Paste into email or text editor
